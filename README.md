# Machine learning from scratch

## Introduction

In this repository, I will gradually implement some freqently used machine learning algorithms from scratch. This would include both the intuitive of the algorithms, and the mathematics bebind. Just ignore the math part if it troubles you. 

This repository will using Python, without any machine learning or deep learning package such as sklearn, tensorflow etc. And we will use [Numpy](http://www.numpy.org/) to facilitate numerical computation. This repository will be written in English, if you find my English is hard to understand, please check the Chinese repository-![中文版](https://github.com/chenxingwei/machine_learning_from_scrach_Chinese/tree/master).
